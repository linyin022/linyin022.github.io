THINKPHP6 常用命令行

Author: Linyin Date: 七月 13, 2022  1:10:49 Category: PHP

tp6框架下常用的终端命令

安装项

安装稳定版框架：composer create-project topthink/think tp

安装视图渲染模板引擎：composer require topthink/think-view

安装多应用模式拓展：composer require topthink/think-multi-app

安装php-jwt拓展：composer require firebase/php-jwt

运行项

启动内置服务：php think run

启动内置服务并且定制端口：php think run -p 端口号

生成项
单应用

创建类库文件（单应用）：php think make:controller Demo

创建数据模型：php think make:model Demo

多应用

自动生成应用（需启用多应用模式）：php think build demo

在index应用中创建一个demo类：php think make:controller index@Demo

在index应用中创建一个student数据模型：php think make:model index@student

在admin应用创建一个Check中间件：php think make:controller admin@Login
